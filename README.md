## 🇷🇸 Srpski - O Projektu
Ovo je **Open Source** motor za portfolio koji automatski generiše HTML galeriju iz foldera sa slikama. Dizajniran je za programere koji žele moderan, "cyberpunk" prikaz svog rada.

### ✨ Karakteristike
- **Dinamičko renderovanje:** JavaScript gradi galeriju "u letu" bez potrebe za ručnim kucanjem HTML-a.
- **Python Automatizacija:** Uključena skripta za kreiranje GIF-ova sa pauzama.
- **Open Source:** Kod je slobodan za korišćenje i modifikaciju.

---

<a name="english"></a>
## 🇺🇸 English - About The Project
This is an **Open Source** portfolio engine that automatically generates an HTML gallery from an image folder. Built for developers who want a sleek, "cyberpunk" display of their work.

### ✨ Features
- **Dynamic Rendering:** Vanilla JavaScript builds the gallery on the fly (No manual HTML editing).
- **Python Automation:** Included script for auto-generating showcase GIFs from static images.
- **Open Source:** Feel free to fork, use, and improve the code.

---

## 🛠️ Stack / Tehnologije
- **Frontend:** HTML5, CSS3 (Custom Neon Theme), Vanilla JS
- **Automation:** Python 3 (imageio library)
- **Hosting:** GitHub Pages
